# AI-Video conferencing System with Real time meeting Transcription.

##  Project Overview
This project is a **full-stack video calling system** that automatically converts spoken conversation into **real-time text**, generates **meeting summaries**, extracts **action items**, and provides a **downloadable PDF report** at the end of each meeting.

The system integrates **WebRTC for real-time video communication** and **AI/NLP techniques** for speech-to-text and meeting summarisation, making it suitable for online meetings, academic discussions, and remote collaboration.

---

##  Objectives
- To build a real-time video conferencing platform
- To convert live speech into text automatically
- To generate intelligent meeting summaries
- To extract key discussion points and action items
- To generate and download meeting transcripts as PDF
- To store and manage meeting history
